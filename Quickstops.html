Quickstops

QuickStops is a web application that allows users to quickly search and save businesses of interest. Optimized for “spur of the moment” use with filtering based on current distance and a clean map UI. Quickstops requires users to create accounts before entering the site and browsings for quick bites and saving them to their favorites. We pulled data from Yelp's Api and refashioned the data to make it easier for the last minute, "of the essence decisions".

Tools and technologies used include: Javascript, React, Node.js. Leaflet, Git, Firebase Hosting, and Firebase Authentication.
