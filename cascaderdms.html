Cascade Bicycle Club RDMS

For my Spring 2020 capstone project, my team worked with the Seattle nonprofit Cascade Bicycle Club as our sponsor to create a scalable solution for their bike locker management. We did struggle initially as our sponsors were largely non technical, keeping all of their data on a single google sheet, so any solution we created had to simple enough for someone who did not know SQL, yet robust enough to generate queries and reports. In the span of 12 weeks, this was the solution we developed:

  • We created a custom relational database management system for them using the Python web framework Django. This comprised of a custom database schema and a database management system for a non technical user to add/change data and do lookups without knowing SQL.
  • We automated their locker maintenance forms and locker sign up forms to autopopulate the corresponding tables when a user fills and submits the form.
  • We created a webscraping tool that would take their existing google sheet and populate the corresponding tables in the new database with the values to migrate their current data.
  • We created a handoff manual and recorded multiple tutorials detailing instructions on how to do common tasks with the new database management system.

Tools and technologies used include, Django, Python, Javascript, React, and AWS.
